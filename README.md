# Country Search Engine App

## Installation

Install my-project with npm

```bash
  npm install
  npm run dev
```

## API Reference

#### Base URL API

```http
  https://restcountries.com/v3.1
```

#### For v2

```http
  https://restcountries.com/v2
```

It doesn't work for get callingcode and currency

## Tech Stack

**Client:** Vue 3, TailwindCSS, Shadcn/ui for Vue
